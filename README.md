# 번개장터 카피

## 프로젝트 설명
번개장터의 모바일의 디자인을 따오고 데이터를 스크래핑하여 판매 상품 데이터를 조회할 수 있는 모바일 웹 애플리케이션입니다.

## 실행
* $ npm install
* $ npm run dev

## 빌드
$ npm run build
